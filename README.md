## Project Description

MyReadings is a web application that allows users to explore and manage a collection of books.

The application supports two different types of users:

- **Readers** → can browse the catalog and view books.
- **Authors** → can add books they have written in addition to browsing the catalog.

The application was developed using **Node.js, Express, EJS and SQLite**.

- **views/** → EJS templates

## Installation

Clone the repository or download the project.

Enter the project folder:

cd MyReadings
Install the dependencies:

npm install

---

## Run the Application

Start the server with:

node app.js

or

npm start

Then open the browser at:

http://localhost:3000
